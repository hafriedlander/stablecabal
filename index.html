The Stable Cabal

noun
  1. A group of people working in loose collaboration to enable the incredible art generating AI called Stable Diffusion.
  2. A set of tools that communicate with a standardised API to allow total freedom to create and explore.

The Server

There's too much to do to reinvent the wheel. All the tools from the Stable Cabal use a common API to connect to a server.

The Stable Diffusion GRPCserver can be run on a machine on your local network, or (coming soon) in Colab or any Docker server.

While many versions of the server exist, if you are unsure then this is the one that we recommend you start with.

For what will be added in the future, a roadmap for future development is available

The Clients

IDEA2.ART

A web UI focused on making generating AI art accessible and easy, while still unlocking all the power Stable Diffusion makes available. You can access it on the web to connect to an existing server, or download it as part of a locally run AIO that combines idea2.art with the GRPCserver in a single easy to install package. The source code is also downloadable.

G-Diffuser

A set of tools including a Discord bot and CLI client focused on exploring the boundaries of Stable Diffusion's capabilities. The discord bot provides an experience familiar to users of Midjourney, and the CLI allows creating matrices that explore the effects of various parameters

Flying Dog Photoshop and Krita plugins

An open source plugin for Krita and a commercial plugin for Photoshop that integrates the Stable Diffusion experience into artist's existing workingflows.

Alternate Servers

Because a common, open protocol is used, users can try out alternative implementations of servers to try new out new features or changes that for one reason or another have not yet made it into the main server.

G-Diffuser Server

In addition to the Discord bot and CLI, G-Diffuser includes a modified server enhanced by it's author Parlance for speed and outpainting quality.

Stability.AI API

Although not part of the Stable Cabal, the Stable Cabal API is an extension to the official Stability.ai API, and many Stable Cabal clients will work (with some limitations due to missing features) with their server.
